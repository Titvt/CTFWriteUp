# 刷新过的图片

## 知识点

`F5隐写`

## 解题

题目提示

> 浏览图片的时候刷新键有没有用呢

半天都没有发现怎么做,看博客才知道用`F5-steganography`

切换到`java8`环境

```
java Extract ./file/Misc.jpg
```

文件输出到`F5-steganography`目录下面

发现`output.txt`

发现`output.txt`是`zip`文件

后缀名改为`zip`

发现用了`伪加密`

![](./img/20-1.png?lastModify=1711202731)

修改加密位后解密,即可获取`flag`
